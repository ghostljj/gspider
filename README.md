## 安装

To start using GJSON, install Go and run `go get`:

```sh
$ go get -u ...
```
```go
  "github.com/axgle/mahonia"  //解决编码用
  "github.com/saintfish/chardet" //自动获取编码用 
  "golang.org/x/net/proxy" //设置代理用
```
此包需要使用到上面几个库，请自行go get -u .... <br/>
支持http(s)代理，Socks5代理 <br/>
